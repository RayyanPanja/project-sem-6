File Naming Conventions:-

filename can be Whatever You Want Just Add suffix.

.view, -- View File
.class, -- Class File
.comp, -- Component File -- not accessible through URL method use Include Or require
Controller  -- Controller File
.ajax -- Ajax File

are applied to use URL Class Method to Access Files...

----------------------------------------------------------------------

Src Folder............................

src/css = filename.css
src/js = filename.js
src/view = filename.view.php
src/view/components = filename.comp.php

Src Folder..............................

----------------------------------------------------------------------

Back_end Folder.............................

back_end/Classes = filename.class.php
back_end/Controllers/* = filenameController.php
back_end/include = filename.php

Back_end Folder.............................
